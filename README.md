### Übersicht
Das ist ein Flutter und Dart Testprojekt mit den Tutos von https://dart.dev und https://flutter.dev, 
das Flutter Projekt ist Birdle, welches ein Pendant zum bekannten Spiel Wordle ist.

### Anwendung
Einfach das entsprechende Release heurnterladen, entpacken und 
für [Linux](https://github.com/realagentluca/Dart_and_Flutter/releases/download/v0.1a/birdle_linux_bundle.7z) die birdle Datei mit
```bash
./birdle
```
starten oder mit Python die [Webseite](https://github.com/realagentluca/Dart_and_Flutter/releases/download/v0.1a/birdle_web_build.7z) hosten:
```bash
python3 -m http.server 8080
```
